同占德骨科诊所 Web 管理系统

一、运行方式
1. 双击 index.html 可直接运行。
2. 也可以在项目目录启动本地静态服务器后访问，例如：
   python -m http.server 8765 --bind 127.0.0.1
   然后打开 http://127.0.0.1:8765/index.html
3. 推荐使用 Edge、Chrome 等现代浏览器。

二、默认账号
管理员：admin / 123456
医生：doctor / 123456
护士：nurse / 123456
药房：drug / 123456
收费员：cashier / 123456

三、权限说明
管理员：全部功能
医生：医生接诊、处方、患者资料、门诊日志
护士：护士登记、检查治疗登记
药房：药品管理、药品发放、药品零售
收费员：门诊收费、报表统计

四、数据保存
本系统是纯前端 localStorage 版本，数据保存在当前浏览器中。
localStorage 键名：tongzhande_clinic_data_v2

维护的数据结构：
patients、visits、prescriptions、drugs、payments、treatments、nurseRecords、users。

五、主要功能
1. 首页仪表盘：今日接诊、今日处方、今日收费、待发药、库存不足、即将过期。
2. 新患者登记：新增、编辑、删除、查询患者。
3. 门诊日志：按日期范围、姓名、诊疗/零售查询，支持 CSV 导出、修改、删除、打印处方单。
4. 医生接诊：今日患者列表、病历录入、常用诊断、常用医嘱、生成处方。
5. 处方开具：添加药品、删除药品、自动计算金额、保存处方、生成收费单、打印处方笺。
6. 处方打印：独立打印样式，标题为“眉县槐芽镇同占德诊所 处方笺”。
7. 检查/治疗登记：X 光、DR、换药、理疗、固定、包扎等项目登记。
8. 护士执行登记：记录执行项目、执行时间、执行护士、执行结果。
9. 药品入库和库存：药品信息维护、库存不足提醒、即将过期和已过期提醒。
10. 药品发放：处方发药后自动扣减库存，库存不足时标记缺药。
11. 药品零售：散客售药、结算、扣减库存。
12. 门诊收费：支持药费、诊疗费、检查费、治疗费、注射费、其他费用和多种支付方式。
13. 患者资料：患者列表、基本信息、历史病历、历史处方、收费记录、检查记录。
14. 报表统计：营业额、利润、药品销售、医生/护士工作量、欠费、退费统计。

六、内置演示数据
系统初始化时内置患者、门诊记录、药品库存、收费记录和李发全处方示例。
如需重置演示数据，进入“数据维护”页面点击“恢复演示数据”。

七、显示适配
界面采用医疗蓝白风格。主工作区、表格、弹窗均支持滚动；表格支持横向滚动；弹窗最大高度受屏幕限制，内容超出时内部滚动。
布局避免大量固定高度，适配 Windows 浏览器缩放 100%、125%、150%。

八、后续扩展建议
当前版本用于业务流程验证。正式使用时建议升级为后端数据库版本，并增加自动备份、操作日志、精细权限、打印机配置和数据导入导出。
